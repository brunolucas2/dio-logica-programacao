# dio-logica-programacao
Exercícios e desafios desenvolvidos no curso de Lógica de Programação da DIO.

## Conteúdo

- **escrevendo-as-classes-de-um-jogo**: classe `Hero` com propriedades (nome, idade, tipo) e método `atacar`, que exibe uma mensagem diferente conforme o tipo do herói (mago, guerreiro, monge, ninja).

- **calculadora-rank-por-vitorias**: calcula o rank de um jogador (Ferro, Bronze, Prata, Ouro, Diamante, Lendário, Imortal) com base na quantidade de vitórias informada.

- **classificador-nivel-heroi**: calcula o nível de um herói (Ferro, Bronze, Prata, Ouro, Platina, Ascendente, Imortal, Radiante) com base na quantidade de XP informada.

## O que aprendi

Durante esses exercícios pratiquei os principais conceitos de lógica de programação:

- **Variáveis e tipos de dados**: como armazenar e organizar informações (nome, idade, XP, vitórias).
- **Estruturas de decisão**: usar `if/else` e `switch` para o programa tomar decisões diferentes dependendo da situação (ex: definir o ataque de cada tipo de herói, ou o rank de um jogador).
- **Laços de repetição**: usar `for`, `while` e `do-while` para repetir ações até uma condição ser satisfeita (ex: pedir dados até o usuário preencher corretamente).
- **Funções**: separar partes do código em blocos reutilizáveis, deixando o programa mais organizado.
- **Classes e objetos**: criar uma estrutura (classe) que representa algo do mundo real (um herói), com suas características (propriedades) e ações (métodos).

## Tecnologias

- JavaScript (Node.js)
